# Hadoop cluster

Move to data directory and run the scripts for :
    
    cd /root/data
    ./scripts/initial_setup.sh
    ./scripts/enlarge_data.sh
    ./scripts/upload_input.sh 
    ./scripts/compile_wordcount.sh 
    ./scripts/run_wordcount.sh
